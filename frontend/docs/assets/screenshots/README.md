## Tips for Good Screenshots

- Use dark mode for consistency (it's the default and looks better in docs)
- Use Devnet with a funded wallet so balances show real numbers
- Create at least one contract before taking dashboard screenshots
- For the audit screenshots, use a real contract with risky clauses so the results look meaningful
- Recommended resolution: 1440×900 or higher, then crop to relevant area
- Use a tool like ShareX, Lightshot, or browser DevTools device mode for consistent framing
